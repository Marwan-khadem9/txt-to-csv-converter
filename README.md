# Text to CSV Converter for EXFOR+ENDF Data

This project provides a Python script that converts text data from an EXFOR+ENDF formatted file into a CSV format. The script is designed to parse specific sections of the input text, extracting relevant data while ignoring comments and formatting issues.

## Features
- Parses EXFOR+ENDF formatted text files.
- Extracts data points including energy values and cross-sections.
- Produces a clean CSV file ready for analysis.
